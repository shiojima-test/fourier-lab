# フーリエ変換ラボ v1.5

スクーミーオレンジボードのセンサー値を Web Serial API で取り込み、ブラウザ上で **「元の波形 → スペクトル → 分解 sin 波」** の 3 つを同時にリアルタイム表示する Web アプリです。

> 「ギザギザの振動 = 綺麗な sin 波たちの合成」をその場で見て体感する授業ツール。

## 公開URL

**https://shiojima-test.github.io/fourier-lab/**

## 何が見られるか

1. **① 元の波形** — オレンジボードから届いた直近 2 秒間の生データ
2. **② スペクトル** — FFT 結果 (周波数 vs 強度)、最大ピークをオレンジで強調
3. **③ 分解 sin 波** — 上位 3 本の sin 波と、それを足し合わせた合成波 (破線)

③ の破線 (合成波) と ① の波形がほぼ同じ形になることが、フーリエ変換の本質体験です。

## 使い方 (生徒向け)

1. オレンジボードに対応スケッチ ([`sketches/`](./sketches/) 参照) を **スクーミーIDE** で書き込む
2. オレンジボードを PC と USB ケーブルで接続
3. **Chrome / Edge** で [https://shiojima-test.github.io/fourier-lab/](https://shiojima-test.github.io/fourier-lab/) を開く
4. 上部のセンサー選択ピル (🔊音 / 📈加速度 / 💡明るさ / 🧲磁気) からスケッチに対応するものを選ぶ
5. 「🔌 USB接続」を押してポートを選ぶ
6. 3 つのグラフが 0.5 秒ごとに自動更新される。センサーを叩いたり光を当てたりして波形を変えてみよう

## 4 種類のスケッチ

| センサー | スケッチ |
|---|---|
| 音 | [sketches/sound.ino](./sketches/sound.ino) |
| 加速度 (MPU6050) | [sketches/accelerometer.ino](./sketches/accelerometer.ino) |
| 明るさ (CdS) | [sketches/brightness.ino](./sketches/brightness.ino) |
| 磁気 | [sketches/magnet.ino](./sketches/magnet.ino) |

すべて 9600 bps、改行区切りで 1 行 1 値を送信します。ブラウザ側はその数値を読み取って 3 つのグラフを描画します。

## 内部仕様 (技術メモ)

- 受信したサンプルは `{value, t (performance.now)}` で直近 5 秒分リングバッファに保持
- 0.5 秒ごとに直近 2 秒分のサンプルを取り出し、平均サンプリング間隔から **実効サンプリングレート fs** を算出
- 2 秒間を fs に合わせて等間隔化 (線形補間) → 平均値を引いて中心化 → Hann 窓 → FFT (Cooley-Tukey radix-2, N = 256/512/1024 から自動選択)
- FFT 結果から上位 3 本のピーク (隣接ビンは局所極大に圧縮) を抽出し、`A_k cos(2π f_k t + φ_k)` で sin 波を再構築
- スペクトルは振幅の最大値で正規化したリニア相対表示。位相・対数軸は表示しない (シンプル維持)

## バージョン履歴

- **v1.5 (2026-05)** ヘッダー右上のボタンを「非破壊検査について」のみに整理 (絵文字・矢印を削除し、控えめな Navy 背景の単一ボタンに統一)。
- **v1.4 (2026-05)** スクーミーが手元にない場面でも体感できる更新。
  - 操作帯に **▶ デモを開始** ボタンを追加。3/6/9 Hz の sin 波 (振幅は時間で envelope 変動・位相は緩やかにドリフト) + 18〜32 Hz の減衰衝撃バースト (5〜10 秒ごとにランダム発生、振幅 5〜15、減衰時定数 0.3 s) + ±1 のホワイトノイズ を約 140 Hz で合成し、通常のリングバッファに流す。実測のような不規則な見た目を出しつつ、スペクトルには 3/6/9 Hz の 3 ピークが残る。USB 接続中はデモ disabled / デモ実行中は USB 接続 disabled の排他制御。デモ中の status は **🟢 デモモード**。
  - 4 グラフすべてに **ガイドカード** (元の波形 / スペクトル / 分解された sin 波 / 合成波と元波形の比較)。各カードに「タイトル」「本文」「何に使う?(青字)」を半透明 cream 背景で配置。
  - 操作帯の **💡 ガイドを表示/隠す** トグル。初期は非表示、デモ開始で自動表示、デモ停止で自動非表示に戻る。それ以外のタイミングではトグルで手動切替。
- **v1.3 (2026-05)** 探究型教材としての完成度を上げた更新。
  - 操作帯に **⏸ フリーズ** ボタンを追加。叩いた瞬間の波形 / 式 / ピーク周波数を画面に固定し、解除でリアルタイム再開 (接続中のみ有効)。フリーズ中もリングバッファへの受信は継続。
  - ③ 分解 sin 波の **個別 ↔ 重ね合わせ表示** を切り替えるトグル。重ね合わせ表示では sin1 (Orange)・sin2 (Blue)・sin3 (Teal) と合成波 (Charcoal 破線) が 1 グラフに並ぶ。
  - 各 sin 波の式 $y = A \sin(2\pi f t + \varphi)$ を **KaTeX** で動的レンダリング (個別行 / 重ね合わせ用の compact 行の両方)。
  - 新規ページ [`nondestructive.html`](./nondestructive.html) — 探究型構成。「課題 → やってみる → 折りたたみ解説」の 5 課題と、応用事例 6 件を折りたたみセクションで掲載。物理式 $f = \dfrac{1}{2\pi}\sqrt{\dfrac{k}{m}}$ は KaTeX 表示。
  - ヘッダーに「🔬 非破壊検査の世界を探究する →」リンクボタン。
- **v1.2 (2026-05)** 数式表示と sin 波の分割表示を導入。
  - 画面上部に KaTeX で `F(ω) = ∫ f(t) e^(-iωt) dt` を表示 (虚数を使った変換式)
  - sin1 / sin2 / sin3 を 1 枚のグラフで重ね描きしていたのを、**縦 3 段の独立ミニグラフ** に分離 (各自の振幅でスケーリング)
  - 各 sin 波に `y = A sin(2π · f t + φ)` 形式の式をリアルタイム表示 (cos→sin 変換で位相を補正)
  - 「合成波 vs 元波形」を独立した ④ グラフに分離。元波形 (Teal 実線) + 合成 (Charcoal 破線) を重ねて比較できる
  - レイアウトを 2×2 グリッドに再構成。フル HD でスクロール無しを維持
- **v1.1 (2026-05)** レイアウト刷新。フル HD でスクロール無しに 3 グラフが全て見える 1 画面構成 (上段: 波形+スペクトル横並び / 下段: 分解 sin 波フル幅)。タイトルから社名と波アイコンを撤去。操作帯にピーク表示を統合し、ガイドメッセージは接続前のみ表示。
- **v1.0 (2026-05)** 初版リリース。音・加速度・明るさ・磁気の 4 センサに対応。リアルタイム波形・スペクトル・分解 sin 波の 3 連表示。
